TCPWM: PWM_PR 
=============
Description
^^^^^^^^^^^
- This example is used to demonstrate generation of random pulse.

Target Device
^^^^^^^^^^^^^
- Traveo-II CYT2B7x devices

CPU Board
^^^^^^^^^
- CYTVII-B-E-1M-176-CPU Rev. C Board

Dependency
^^^^^^^^^^
- Check the Port 6.1
   Port 6.1: BB_LIN3_TX

Expectation
^^^^^^^^^^^
- Check each PWM port, it will output random pulse.
