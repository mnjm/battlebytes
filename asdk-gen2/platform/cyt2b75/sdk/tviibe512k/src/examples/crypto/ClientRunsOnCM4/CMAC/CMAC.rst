Crypto: CMAC (ClientRunsOnCM4) 
==============================
Description
^^^^^^^^^^^
- This example is used to demonstrate CMAC Test. Calculate CMAC with 128bit key and compare it with reference value.

Target Device
^^^^^^^^^^^^^
- Traveo-II CYT2B7x devices

CPU Board
^^^^^^^^^
- CYTVII-B-E-1M-176-CPU Rev. C Board

Dependency
^^^^^^^^^^
- None

Expectation
^^^^^^^^^^^
- Calculate CMAC with 128bit key and compare it with reference value.
